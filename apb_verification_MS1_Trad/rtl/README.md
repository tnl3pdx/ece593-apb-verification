# RTL Sources

Design files for MS1.

This directory includes the APB source files with our custom APB peripheral.

Note: As of MS1, our custom peripheral is still work in progress. Once our peripheral is done, it will be moved into the APB folder.

# Ownership Note

The RTL that is being used for this project comes from tom-urkin's AMBA repository (https://github.com/tom-urkin/AMBA). We are only using the source code from their APB system, but we will be designing our own testbench and eventually UVM. We are planning on implementing our own APB peripheral to add some originality to the source code.
